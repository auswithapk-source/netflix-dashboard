 # Netflix data analysis  (Interactive dashboard using Power BI)
 # Project objective
 To analyze the distribution of content by type (movies vs. TV shows), genre, and country of origin.
 # Questions
 What are the most popular genres for movies and TV shows?
 Which countries produce the most content on Netflix?
 How has the distribution of content (movies vs. TV shows) changed over time?
 What is the average rating or popularity score for content in different genres or countries?
